MT6589之Enginner Mode(工程模式)

Camera Camera-tuning 2018/10/22

1.MTK Camera Enginner Mode的功能（MTK平台的工程模式）

  • Change Capture Setting（更改拍照的设置）

  • Capture Raw Data（捕获Raw图的数据）

  • AF Test（AF测试）

2.Camera Engineer Mode(工程模式)进入方法

第一步：输入以下命令

第二步：进入Enginner Mode，向左滑动屏幕到"Hardware Testing"选项

第三步：选择Camera，进入Camera Enginner Mode（工程模式）

第四步：设定完成后，按下"Preview"，进入预览

第五步：按下画面中的"Capture"，进行拍照

3.Capture Setting

3.1.ISO Speed

ISO Speed：选择拍照时的ISO

  • Manual ISO（手动设置ISO）

  • ISO Auto

3.2.Flicker

Flicker：选择Flicker Mode

  • 50HZ：曝光时间必须是10ms的整数倍

  • 60HZ：曝光时间大约是8.33ms的整数倍

3.3.Strobe Mode

Strobe Mode： 选择拍照的模式，是否开启闪光灯

  • Auto

  • On

  • Off

4.MTK Enginner Mode功能之Capture Mode

Capture Mode可以拍摄Raw图数据，用于提供MTK分析

4.1.Capture Size

Capture Size：选择拍照时的Resolution（分辨率）

  • Preview Size

  • Capture size

  • Video size

4.2.Capture Type

Capture Type：选择拍照时存Raw或是Jpeg

  • Processed Raw：经过OB，Shading的Raw Data

  • Pure Raw：Sensor直接输出的Raw Data

  • Jpeg Only

4.3.Capture Number

Capture Number：选择拍照时，一次存几张照片

  • Single

  • Double

  • Triple

5.MTK Enginner Mode功能之AF Test

在Enginner Mode中，可以执行各种AF模式，测试Lens Module。

5.1.Auto

Auto：拍照时，使用Normal AF。

5.2.AF Bracket

AF Bracket：拍照时对Final AF Pos进行Bracket Capture。=>也就是说，拍照的时候先做一次Normal AF，Normal AF做完之后，对Final AF Position进行一个Bracket Capture的动作，Bracket Capture是指它会针对Final的Position（位置）的周围的AF Position都各拍一张。

  • 选择Bracket范围：、0~511

  • 选择Bracket Capture间隔：1/2/4

举例：如果我们最后拍AF Final Position是500步，假如Bracket的范围选择是10，Bracket Capture间隔是1的话，那么就会从500步这个位置减去10步的位置就是490步，加上10就是510步；也就是说从490步开始每间隔1步拍一张照片一直拍到510步，

5.3.Full Scan

Full Scan：拍照时，使用全域搜索AF。=>也就是说，在拍照的时候每隔一步就去搜寻一次，它会把整个AF的区域搜索一次，最完整的搜寻方法，产生AF Table的搜寻方式。

5.4.Through focus dump

Through focus dump：拍照时，每隔特定Focus Interval拍一张照片

  • 选择搜寻间隔：0~511步

  • 选择搜寻方向：（Inf to Marco（远焦到近焦）/Marco to Inf（近焦到远焦）/Manual（手动做选择））
