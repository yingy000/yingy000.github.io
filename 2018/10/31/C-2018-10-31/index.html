MT6589_Advance之Debug Parser

工具 Debug Parser 2018/10/31

1.目的

查询图片中的Camera Info（信息）

2.提供功能

3.ISP Info

3.1.ISP Index

ISP Index：显示ISP的index（段数）

De-mosaic会根据不同的ISO去决定要使用哪一组De-mosaic，所以会有不同的Index。

3.2.ISP Value

ISP Value是指3*3Color Correction Matrix（色彩校正矩阵）

第一行表示对R处理，第二行表示对G处理，第三行表示对B处理……

G2G_CNV_00~G2G_CNV_22均为正数，表示需经过转换，才能正确显示负数的Element，转换公式如下所示。

3.3.ISP Slider Position

ISP Slider Position：显示ISP Slider的Position（强度）

CCT可以调试ISP的一些Noise、Texture的强度，也就是说CCT用Slider的方式去调整ISP的强度，CCT调整完这些强度同时它也会记录在图片里面，这时可以利用DebugParser来读出这张照片使用ISP的处理强度是多少，如果说目前的照片不够锐利（Sharpness不足），就可以通过DebugParser读出一些相关的Slider强度是不是已经调到了极限值，如果没有调到极限值，就意味着可以利用CCT继续增大图片的锐利度，反之，如果已经调到了极限值，那就没办法再增大了，只能通过其他的途径来增大图片的锐利程度。

7.AWBV

7.1.第一个功能

显示AWB算法判断画面中各个色块的色温

点击按钮[AWBV]，然后选择按钮[Light Source],就会显示AWB算法判断画面中各个色块的色温。

假如说当前的光源是CWF(绿色)，那么当前中心色（灰色）的部分也一定是绿色的，可以辨认出画面中的中心色是否掉到绿色的Light Source Area。

7.2.第二个功能

显示AWB算法判断画面中各个色块白点的落点

点击按钮[AWBV]，然后选择按钮[XY Distribution],会显示AWB算法判断画面中各个色块的色彩和色温。
