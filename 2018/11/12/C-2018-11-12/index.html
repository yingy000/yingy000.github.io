CCT6589_Basic之CCM Tuning

工具 CCT 2018/11/12

1.简单了解

CCM（Color Correction Matrix）是指MTK提供的RGB Domain校正Color Correction的方法。

CCM为3*3的Matrix，目的为改变色彩饱和度与降低整体Color Error。

正常的CCM并不会改变中性色，也就是说，灰色经过CCM还是灰色。

2.调试步骤

1.Import Target（可以选择CVS or Jpeg）

2.Import Gamma（目前使用的Gamma）

3.Import Source（Pure Raw）

4.Saturation（选择Saturation）

5.Dynamic CCM tuning

6.Manual AWB Gain

7.DNG Setting

8.Save to NVRAM（存入NVRAM）

2.1.Import Target（可以选择CVS or Jpeg）

Import Jpeg后，将24色框选中，然后按Auto Select

Tool会询问是否重新选取，若已经正确选取，点NO

2.2.Import Source

Import Raw后，将24色框选中，然后按Auto Select

Tool会询问是否重新选取，若已经正确选取，点NO

2.3.Dynamic CCM tuning

2.4.改变饱和度

改变饱和度，重新Optimize

饱和度越小，CCM的对角线数值越小

2.5.设定COST Func

设定最佳化的方程式

2.6.设定Constrain

设定最佳化的条件

可以设定Hue和Chroma

3.两种手动微调的最佳化方式

3.1.拖曳Target

拖曳Target到期望的Lab Domain位置

注意

无法保证CCM一定能使结果趋近新的Target

指定色块接近原本Target后，可能导致其他色块远离原本target

3.2.设定Constrain

针对特定的Patch，设定Constrain，可以限制Result和Target的距离在一定的范围之内。

Constrain可设定Hue Range，Chroma Range，设定完成后按Set，然后再下方的Enable Patch勾选

注意

无法保证CCM一定能使结果落在Constrain中，只能尽量趋近

指定色块接近Target后，其他色块可能远离原本Target
