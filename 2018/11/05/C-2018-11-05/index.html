2.1.混光亮度权重调整

2.1.1.意义

当环境色温为多组光源混合而成，AWB会根据此table，决定各色温的比重

2.1.2.修改时机

当AWB判断光源错误，可修改预设权重，降低错误光源所占的比例

2.1.3.举例

当在Daylight环境，环境亮度为LV12，且照片发红：

  • 1.由DP得知AWB最终机率

  • 2.由DP得知Shade在LV12混光权重为66，修改Table，降低Shade权重。

2.2.单一光源和日光亮度权重调整

2.2.1.意义

各种光源都有合理的亮度范围，当光源亮度超过此范围，则WB Gain和Daylight Default Gain做线性混合

2.2.2.修改时机

当AWB光源判断正确，欲又发生色偏，可确认是否为Daylight Default Gain造成

2.2.3.举例1

2.3.AWB收敛速度调整

2.3.1.意义

决定AWB的收敛速度和灵敏度

2.3.2.修改时机

当AWB收敛速度过慢或对于环境光源变化的灵敏度过低

2.3.3.相关参数

iSpeed：AWB每Frame收敛的百分比，设越高收敛越快

i4StableThr：AWB判断是否收敛的灵敏度，设越小越灵敏
