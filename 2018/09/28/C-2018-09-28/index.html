CCT6735M的了解与使用-AWB Tuning

工具 CCT 2018/09/28

1.AWB Tuning

1.1.MTK CCT AWB Tuning提供的功能

1.Module AWB Light Source Area Tuning(模块AWB光源区域调优)

新模组必须重新校正模组白点在各个色温下的落点，AWB结果才会正确。

2.AWB Preference Tuning(AWB偏好调优)

CCT可用Slider调整控制各个色温的喜好度色偏（参考3.1），比如黄光下（暖光场景）可以偏黄而不是全白，蓝天更蓝，白云更白。

各种色温下也可以给予一组Preference Gain，Preference Gain是指乘上这个Gain之后，就会白平衡，比如说给D65光源乘上一个比较偏红的Gain值的话，这时就算是拍灰卡，拍出来的灰卡的照片也会是偏红黄的…

如果不管怎么tuning一些特殊场景都不能达到白平衡的时候，这时加上一组Preference Gain，但是这就牺牲掉了正常场景的Preference以此来修正一些特殊场景。

  • Tungsten/Warm Fluorescent/Shade：同时提供Slider&Preference Gain

  • Daylight/CWF/Fluorescent/Daylight Fluorescent：仅提供Preference Gain

    DF指的是Daylight(日光) Fluorescent(荧光)，Tungsten（T光），Horizon（H光），Warm Fluorescent，Shade代表D75光源

2.CCT AWB Tuning Flow

1.在Common Control Dialog中，按下"AWB Default"

按下"AWB Default"按钮之后，默认会有以下动作：

  • 将AWB Disable，并设为1x Gain

  • 关闭CCM/PA/Saturation

2.指定Capture Setting

将Format设为Raw Ext=>需要使用Raw图来tuning AWB的效果

Mode设为Capture

3.指定正确的Shading Table

  • 当拍摄D65/D75/DNP时，选用高色温（高色温的Shading Table）

  • 当拍摄CWF/TL84时，选用中色温（中色温的Shading Table）

  • 当拍摄A/Horizon光时，选用低色温（低色温的Shading Table）

4.拍照（灰卡）后在图片中框选一块ROI(感兴趣的区域)，然后填入对应的Light Source中。

5.反复进行步骤2、3、4填入八个色温的Data，这时就可以点击"Auto Tuning"。

6.按下"Auto Tuning"让Tool自动产生AWB Light Source Area(AWB光源区)。

注意：如果之前已经Tuning完成，且将Data存在手机中，想再次确认资料的正确性，可以使用”NVRAM Tuning”，利用已经存在NVRAM中的资料，将直接产生AWB Light Source Area，就不用再次复测，重复上面的那些步骤。

7.点选AWB Light Area，可看到Tool Auto Tuning的AWB结果。

AWB Light Source Area

从图中可以看到有6个AWB Light Source Area，我们会观察灰点（中心色的点）在这6个光源下它的RGB的值会落在哪个范围。

8.点选PWB Light Area，可看到Tool Auto Tuning的PWB结果

PWB Light Source Area(MWB)

可以选择在不同光源（日光、荧光、夜景等）下的设定

3.Preference Tuning(偏好调试)

3.1.提供三条Silder进行特定光源下色偏程度的控制

  • Slider越大越色偏，Slider越小越白平衡

  • Tungsten代表Horizon(H光=低色温)

  • Warm Fluorescent代表A光

  • Shade代表D75

3.2.提供Preference Gain，进行各种光源下Preference Gain的设定

  • 建议不要轻易使用Preference Gain，AWB会将运算结果强制乘上Preference Gain。

  • 可能为了解特定场景偏色，而导致其余场景都偏色。

  • 最恰当的使用时机，若AWB始终无法白平衡，且都偏相同色调。

4.实际应用举例

4.1.问题一

某些室内日光灯，AWB结果偏绿，且白点落在Daylight下方，该如何处理？

答案：首先需要确认在无法白平衡时它的中心色的落点是落在哪里的？这时可以在该光源下拍一张灰卡的Raw图，选择感兴趣的区域（灰色的区域），然后在CCT中看它的灰色的点会落在哪个部分，通常如果偏绿的话，代表这个灰色的点没有在定义的AWB Light Source Area中，如果说落在Daylight下方，说明就是特殊的光源（Daylight Fluorescent Light），这时候不建议去放大Daylight的这个方框，不然会产生其他影响，建议增加AWB Daylight Fluorescent Light Source Window，也就是增加一个Daylight Fluorescent的框（特殊光源的框），以此把这个灰点（中心色的点）包起来。

4.2.问题二

当Daylight(日光)出现AWB偏蓝，该如何处理？

答案：首先同样需要确认在无法白平衡时它的中心色的落点是落在哪里的？这时可以在该光源下拍一张灰卡的Raw图，选择感兴趣的区域（灰色的区域），然后在CCT中看它的灰色的点会落在哪个部分，如果偏蓝说明Daylight的框没有框住，这时可将Daylight AWB Window（Daylight的框）往右扩大

  • PS：当特定光源偏蓝，将对应的Window往右移

  • 偏红，往左移

  • 偏紫，往上移

  • 偏绿，往下移
